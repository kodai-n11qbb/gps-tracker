# gps-tracker
## 概要
移動体(ラジコン等)にGPS機器を取付け、訓練用教材とする

raspberrypiと同じwifi内に接続し、localhost:7777 に接続すると現在地が表示される

だけ

## 使用機器・環境
AE-GPS\
raspberrypi zero 2 W\
raspberrypi OS legacy 32bit
